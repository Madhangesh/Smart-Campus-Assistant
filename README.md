# AI-Powered Smart Campus Assistant

An intelligent learning assistant built using Streamlit, spaCy, Sentence Transformers, and PyPDF.

## Features
- Upload PDFs
- Semantic Q&A
- Multi-level summarization
- MCQ quiz generation
- Learning analytics
- Premium UI

## Tech Stack
Python | Streamlit | spaCy | Sentence Transformers | NLP

## Run
streamlit run app.py
